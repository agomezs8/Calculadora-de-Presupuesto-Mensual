Calculadora de Presupuesto Mensual

Este repositorio reúne toda la documentación creada para el proyecto “Calculadora de Presupuesto Mensual”. Aquí se incluyen los requerimientos del sistema, el plan de pruebas, la propuesta de mantenimiento, la investigación sobre Markdown y las evidencias del trabajo realizado. El objetivo principal del proyecto es practicar la documentación técnica y el uso de GitHub como herramienta de control de versiones.

Contenido del repositorio

Requerimientos: contiene dos versiones del documento DRS (v1 y v2), donde se describen el propósito del sistema, sus funciones y las mejoras añadidas.

Pruebas: incluye el plan de pruebas funcionales, con tres casos que validan el registro de datos y el cálculo del balance.

Mantenimiento: aquí se encuentra la propuesta de mantenimiento correctivo, adaptativo y perfectivo.

Investigación: incluye un archivo en Markdown donde se explica qué es Markdown, para qué sirve y por qué es útil en proyectos de software.

Evidencias: contiene las capturas del historial de commits, del versionado de documentos y de la estructura del repositorio.

Qué se hizo en este proyecto

Se creó la documentación del sistema, se versionaron los archivos en GitHub y se llevaron registros claros mediante commits. También se generaron documentos en Word y Markdown, siguiendo una estructura organizada. El control de versiones permitió mantener todo ordenado y ver cada cambio realizado durante el proceso.

Este repositorio resume todo el trabajo realizado para completar el proyecto y muestra cómo se usó GitHub para documentar, organizar y respaldar cada parte del desarrollo.
